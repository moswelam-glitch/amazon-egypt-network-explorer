🏪 HCD3 — Amazon Now (Dark Stores)

Ultra-fast delivery ecosystem achieving 13–17 minutes from order to doorstep. Operates through dark stores (400–500 sqm, 8 associates/shift) fed by the Regional Distribution Center, using a pick-to-ship model with no separate packing step.

📦 SPX9 — Storage Distribution Center (SDC)

External storage backbone for Fulfillment Centers handling items with vendor lead times exceeding 21 days. Feeds replenishment inventory to HCD3 dark stores and CAI6 on a scheduled, forecast-driven cadence.

FC — Inbound Operations

Dock → Receive → Stow. Each item progresses through defined system states before becoming available for picking.

1 DOCK

Entry point for all vendor shipments. Associate opens Dock Master to monitor status: Scheduled → Arrived → Closed.

Gate Check-In: Truck verified for timeliness. Late arrivals flagged.

Yard & Dock Assignment: TDR must be set before unloading begins.

Unloading: Dock ambassador scans every item. System accepts shortages but NOT excess.

Confirmation: Vendor receives signed document. Shipment moved to NYR storage.

2 RECEIVE

Items formally inducted into inventory. Associate determines receive type: Each | Prep | Case | Pallet.

Product Scanning: Validates dimensions, weight, specs in real time.

Expiry Check: For applicable categories. No visible expiry → escalated to Problem Solve.

Trolley Placement: Status changes to NYS (Not Yet Stowed).

3 STOW

Items physically placed into designated storage locations.

Location Selection: System recommends based on item height/weight.

Rules: BOAATS Rule (bin organization) • ASIN Limits Per Shelf

Reactive Stowing: Priority mode for SPX5 transshipments destined for active orders.

Completion: Item status → Available (live and pickable).

FC — Outbound Operations

Pick → Wrangle → Sort → Pack → Dock. Orders move from storage to customer shipment.

1 PICK

Associate assigned a picking path/batch. System minimizes walking distance.

Process: Directed to location → scan location → pick → scan item → place in tote/cart → conveyor to Wrangle.

2 WRANGLE

Buffer stage. Totes scanned and grouped by batch between Pick and Sort.

3 SORT

Multi-item orders organized by customer using a rebin grid. Each slot = one customer order.

4 PACK

System suggests optimal packaging. SLAM: Scan, Label, Apply, Manifest — generates shipping label.

5 DOCK (Outbound)

Pallet building → shrink wrap (required) → load onto correct trucks → manifest closed, departure recorded.

Middle Mile — Sorting Center

Central sorting and reverse logistics hub. 2-hour SLA from receipt to outbound lane.

Facility Profile

~7,000 cubic meters

  • Dual-lane conveyor (Flyers + Boxes)
